How to run this script

1.Download the zip file

2.Extract the file and copy "SyntaxSquadFinal" folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Start XAMPP etc Mysql.

5.Open PHPMyAdmin (http://localhost/phpmyadmin). Import tab-> import library_management_system.sql, this will create tables and db.

6. Run the script http://localhost/SyntaxSquadFINAL/

7. from homepage -> register user->login -> borrow books or edit your profile

8. Can also login as admin accounts that we created from db.
